# Monstah Plant Detector: App Summary & Uses

## 🌿 App Summary
**Monstah Plant Detector** is a premium, AI-powered React web application designed for real-time plant identification and health diagnosis. It leverages a modern Full-Stack architecture to provide a seamless, interactive user experience.

*   **Frontend**: Built with **React 19** and **Vite**, featuring **Framer Motion** for smooth transitions and a custom **Web Audio API** sound system.
*   **Infrastructure**: Uses **AWS S3** for persistent cloud image storage and **Vercel Serverless Functions** for backend logic.
*   **Intelligence**: Integrates **OpenAI (GPT-4o Vision)** to analyze photos, identify species, and assess health.
*   **Storage**: Hybrid of **Cloud (S3)** and **Local (Browser localStorage)**.

---

## 🛠️ Core Uses & Features
1.  **Instant Plant Identification**: Identify common and scientific plant names from photos.
2.  **AI Health Diagnosis**: Get a numeric health score and status badge (Excellent/Good/Fair/Needs Attention).
3.  **Care Recommendations**: Access watering, light, soil, and fertilizer tips.
4.  **Personal Plant Library**: Save results to a local dashboard for later reference.
5.  **Cloud Storage Integration**: Automatic S3 uploads and clean-up.
6.  **Premium UI/UX**: Custom sound effects and modern animations.
